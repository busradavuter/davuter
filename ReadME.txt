ToDoList Application

To Run the Application,

The Java project is located in the restapi folder. Open this project with Netbeans
Create a mysql database named 'reactapi'.
In the restapi project, set your database settings from the application.properties section in the resource section.
Do not change of "server.port = 9090". Frontend works on port 9090.
Right-click on the project and make "RunAs > Project Clean Build"
Right click on com.davuter.restapi > reactapiApplication.java and run RunAs> Java Application.
Start the index.html file in the restapiPage folder in your browser after the project starts
